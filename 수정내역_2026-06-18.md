# 신용등급 관리 시스템 — 수정내역 (2026-06-18)

> 서버 실행: 프로젝트 폴더에서 `python app.py` → 브라우저 http://127.0.0.1:5000

## 이번 작업에서 변경된 내용

### 1. 등급 색상 범례 삭제
- 표 하단 범례에서 `등급: ●AAA~AA ●A ●BBB ●BB ●B 이하` 색상 안내 제거
- `등급유형` / `변경` 안내는 유지
- 파일: `templates/index.html`

### 2. 미래에셋증권 로고 삭제
- 상단 좌측 로고 이미지 제거 → "신용등급 관리 시스템" 제목만 표시
- 파일: `templates/index.html`

### 3. 등급 상이 표시 (최종등급 근거)
- 3사(나이스·한국기업평가·한국신용평가) 등급이 서로 다르면
  "⚠ 등급상이건 있음" 메시지와 **상이한 평가사 이름** 표시
- 판정: 소수(아웃라이어) 평가사를 상이로 봄. 3사 모두 다르면 3사 전부 표시
- 파일: `app.py` (`build_row`의 rating_mismatch/mismatch_agencies), `templates/index.html`

### 4. NICE 매칭 버그 수정 (중요)
- 증상: `KB라이프생명보험`이 **신한라이프생명보험의 AAA**를 잘못 가져옴
- 원인: 영문 접두사(KB) 제거 별칭 `라이프생명보험`이 `신한라이프생명보험`에 부분일치
- 수정: `_cell_matches_name()` 헬퍼 추가 — 별칭이 셀 이름의 접두부일 때만 매칭(다른 브랜드 접두사 거부)
- 파일: `scraper.py`
- 데이터 정정: `data/ratings.json`의 KB라이프생명보험 나이스 칸 비움(최종등급 AA+ 유지)

### 5. 엑셀 다운로드 기능 추가
- 각 업권 섹션 헤더 우측에 「⬇ 엑셀 다운로드」 버튼
- 클릭 시 **전체 기관(80건)**을 첨부 양식대로 한 파일로 다운로드
- 양식 파일: `export_template.xlsx` (원본: 바탕화면 `다운로드파일양식.xlsx`)
- 컬럼: 업권 | 기관명 | 나이스신용평가 | 한국기업평가 | 한국신용평가 | 최종적용신용등급 | 최종등급 근거
- 업권 표기: 증권사 → `증권` (엑셀에서만)
- 파일: `app.py` (`/api/export.xlsx`), `templates/index.html`
- 의존성: `openpyxl` (설치됨)

### 6. admin.Goun 표시
- 표 하단 범례 줄 오른쪽 끝에 주황색 `admin.Goun` 표시 (화면 고정 아님)
- 파일: `templates/index.html`
